# ⚽ EPL 2026/27 Predictor

**Be Competitive · Support Your Club · Outsmart Your Friends**

AI-powered Premier League prediction app with a season-long sweepstake for you and your mates.

🔗 **Live:** [epl-2027.vercel.app](https://epl-2027.vercel.app)

---

## 🎯 Features

### Sweepstake
Create a private room, share the code, compete all season.
- Pick your **2026/27 title winner**
- Predict **match scorelines** before kickoff
- Live **leaderboard** updates automatically
- **Points:** Exact score = 5pts · Correct result = 2pts · Right champion = 10pts

### Predictions & Analysis
- **🔮 Predict Match** — Poisson distribution model + AI tactical preview for any fixture
- **⚡ Power Rankings** — Squad strength ratings for all 20 clubs
- **🔍 Team Intel** — Per-club deep dive with AI season preview
- **📊 Results Tracker** — Track how accurate the model is over the season

### Live Data
- **🔴 Live Scores** — Real-time EPL scores, auto-refresh every 60s
- **🏆 Season Table** — Live standings with European/relegation zones
- **🥅 Top Scorers** — Golden Boot race

## 🛠️ Tech Stack

- **React 18** + **Vite**
- **Poisson distribution** match prediction model
- **Anthropic Claude API** — AI previews with live web search
- **ESPN API** — live scores, standings, top scorers (no key needed)
- **JSONbin.io** — sweepstake room storage
- **PWA** — installable on iOS and Android
- Deployed on **Vercel**
